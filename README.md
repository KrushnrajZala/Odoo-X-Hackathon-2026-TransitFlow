# TransitOps - Smart Transport Operations Platform

## 🚀 Overview
TransitOps is a comprehensive transport operations management platform designed to digitize and streamline vehicle, driver, dispatch, maintenance, and expense management.

## 🎯 Features
- **Authentication & RBAC**: Secure login with role-based access control
- **Dashboard**: Real-time KPIs and operational insights
- **Vehicle Management**: Complete vehicle lifecycle management
- **Driver Management**: Driver profiles, licenses, and safety scores
- **Trip Management**: End-to-end trip creation and tracking
- **Maintenance**: Vehicle maintenance scheduling and tracking
- **Expense Management**: Fuel and operational expense tracking
- **Analytics**: Fleet utilization, efficiency, and cost analysis

## 🛠️ Technology Stack
- HTML5, CSS3, Bootstrap 5
- JavaScript (Vanilla)
- PHP 7.4+ (Core PHP)
- MySQL with MySQLi (Prepared Statements)
- Font Awesome 6
- Chart.js
- SweetAlert2
